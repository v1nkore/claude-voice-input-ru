# Русский голосовой ввод для Claude Code

## Способ 1 — встроенная диктовка Claude Code (уже настроена)

В `%USERPROFILE%\.claude\settings.json` включено:

```json
{
  "voice": { "enabled": true, "mode": "hold" },
  "language": "russian"
}
```

Использование: **перезапустите Claude Code**, затем **зажмите пробел** в пустой строке ввода — говорите по-русски, отпустите — текст появится. Команда `/voice tap` переключает в режим «нажал — говоришь — нажал ещё раз».

Если микрофон не работает: Параметры Windows → Конфиденциальность и безопасность → Микрофон → включить «Доступ к микрофону для классических приложений».

## Способ 2 — локальный Whisper (этот скрипт, работает в любом окне)

Запустите `start_voice.bat` (или `python voice_input.py`).

- **F9** — начать запись (высокий сигнал)
- **F9** ещё раз — остановить, распознать и вставить текст в активное окно
- Распознавание полностью офлайн; при первом запуске скачается модель
  (по умолчанию `large-v3-turbo`, ~1.6 ГБ — лучшее качество для русского)
- Модель настраивается переменной окружения `VOICE_MODEL`
  (`tiny` / `base` / `small` / `medium` / `large-v3-turbo`); на слабой машине:
  `setx VOICE_MODEL small`. Там же: `VOICE_HOTKEY`, `VOICE_VOLUME`, `VOICE_TARGET_WINDOW`

Установка зависимостей (на новой машине, нужен Python 3.10+):

```
pip install -r requirements.txt
```

### Автомьют Discord на время записи

F9 регистрируется как системная горячая клавиша эксклюзивно (`RegisterHotKey`) — если
в Discord тоже назначен F9 на переключение микрофона, он перестаёт срабатывать в
Discord (побеждает тот, кто зарегистрировался первым). Чтобы не терять функцию мьюта:

1. Открой Discord → нажми на шестерёнку рядом со своим ником внизу слева
   (Настройки пользователя) → слева в списке раздел **«Голос и видео»** →
   там же (или отдельным пунктом) **«Горячие клавиши»**.
2. Найди строку про отключение/переключение микрофона (обычно называется
   «Переключить откл. микрофона» или «Mute») — нажми на поле с текущей клавишей
   и вместо F9 нажми **F6** на клавиатуре (обычная клавиша в верхнем ряду,
   шестая после Esc). Сохранится автоматически.
3. Готово — скрипт сам шлёт F6 в начале и в конце записи, так что микрофон
   в Discord глушится, пока вы диктуете, и включается обратно, когда закончили.

Если не находите нужный пункт в настройках — пришлите скриншот раздела «Горячие
клавиши», разберёмся вместе.

Отключить: `VOICE_MUTE_DISCORD=0`. Сменить клавишу: `VOICE_DISCORD_MUTE_KEY=...`.

## Способ 3 — встроенная диктовка Windows

**Win + H** в любом поле ввода. Язык распознавания = язык раскладки клавиатуры: переключитесь на русскую раскладку (Win+Пробел) и диктуйте. Если не работает — Параметры → Время и язык → Речь → добавить русский.
